Using one image for the pattern and a sprite for ornaments.

  • Font: Francois One
  • Images: two
    • A 3x6 tiled image for the pattern:
    • A 200x160 sprite for all ornaments
  • Pro: gradients via CSS3, minimum http requests, two if an ornament is needed.
  • Pro: we can embed html encoded symbols without images.
  • Pro: can extend or stretch.
  • Con: ie7 users get a degraded version, no ornaments.
  • Con: (ideally) all ornaments must have the same size, in this case 20px x 20px.
