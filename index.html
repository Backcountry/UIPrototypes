  • factoids
  • Font: Francois One
  • Images:
    • A 3x6 tiled image for the pattern:
    • A 200x160 sprite for all ornaments
  • Pageload: 46.4kb | font: 30.6kb | css: 11.6kb | html: 2.7kb | images: 1.5kb
  • pros
  • Uses gradients via CSS3, minimum http requests (9 total in this page)
  • We can embed html encoded symbols without images.
  • We can extend or stretch.
  • In order to achieve similar quality using images you would need at least an 79,6kb JPG file and six additional http requests.
  • cons
  • ie7 users get a degraded version, no ornaments.
  • (ideally) all ornaments must have the same size, in this case 20px x 20px.
